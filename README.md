### Projects

**🔒 [SDN-based firewall](https://github.com/Nishant-Hegde/sdn-firewall)**
`Python` `OpenFlow` `Mininet` `Open vSwitch`
> Controller-based firewall using POX + Mininet. Intercepts `packet_in` events, performs MAC learning, and installs OpenFlow DROP rules directly on the switch for blocked IP pairs — with structured timestamped logging.

---

**🖧 Secure seat reservation system**
`Python` `TCP Sockets` `SSL/TLS` `Multithreading`
> Multi-client distributed reservation server with thread locking to prevent double-booking under concurrent load. SSL/TLS encrypted comms with self-signed certs and persistent JSON-based storage.

---

**🍽️ CanteenX**
`React` `TypeScript` `Node.js` `MongoDB`
